---

### ⚡ Advanced Features

| Action Type | Description |
| :--- | :--- |
| **📱 URL to APK** | যেকোনো ওয়েবসাইট ইউআরএল-কে সরাসরি একটি সিকিউর অ্যান্ড্রয়েড অ্যাপে (APK) রূপান্তর করে। |
| **⚛️ React to APK** | আপনার রিঅ্যাক্ট প্রজেক্ট থেকে অটোমেটিক বিল্ড তৈরি করে সরাসরি অ্যান্ড্রয়েড অ্যাপে কনভার্ট করে। |
| **📦 Unzip & Build** | জিপ ফাইল এক্সট্র্যাক্ট করে প্রজেক্ট নরমালাইজ করে এবং `tnx_bd` ব্রাঞ্চে পুশ করে। |
| **🧹 Clean Repository** | রিপোজিটরির অপ্রয়োজনীয় ফাইল মুছে ফেলে একদম ফ্রেশ সেটআপ তৈরি করে। |
| **🏗️ Build Only** | বর্তমান কোডবেস থেকে প্রোডাকশন লেভেল বিল্ড তৈরি করে ডেপ্লয় করে। |

> **ধাপ ১:** উপরের **Open Actions** বাটনে ক্লিক করুন।  
> **ধাপ ২:** বাম পাশের লিস্ট থেকে **"Tnayem48 Universal Runner"** সিলেক্ট করুন।  
> **ধাপ ৩:** **Run workflow** ড্রপডাউন থেকে আপনার পছন্দের **Action** টি বেছে নিন।  
> **ধাপ ৪:** রান শেষে **Artifacts** সেকশন থেকে আপনার APK ফাইলটি ডাউনলোড করে নিন।
